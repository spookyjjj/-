# block요소
- `div`
- `ul` `ol` `li`
- `p` `h1`
- [O] width, height, margin, padding
- [X] vertical-align

# inline요소
- `a`
- `img`
- `span`
- `strong`
- [O] vertical-align, line-height 
- [X] width, height
margin, padding은 좌우로만 반영! 상하로는 반영되는것 처럼만 보임  

# inline-block요소
- `button`
- `input`
- `select`
